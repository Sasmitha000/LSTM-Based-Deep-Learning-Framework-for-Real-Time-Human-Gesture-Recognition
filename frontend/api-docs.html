API Documentation

SignBridge provides a REST API and WebSocket interface for real-time sign language detection and translation. This documentation covers all available endpoints, request formats, and response structures.

🔌 WebSocket Interface

Connection

Connect via Socket.IO for real-time gesture detection. Send video frames and receive predictions instantly.

const socket = io('https://lstm-based-deep-learning-framework-for-real-time-production.up.railway.app');

Emit: frame

Send a base64 encoded video frame for processing.

socket.emit('frame', { frame: 'data:image/jpeg;base64,/9j/4AAQ...' });

Listen: prediction

Receive real-time prediction results after each frame.

socket.on('prediction', (data) => { // data.gesture — detected gesture name or empty string // data.confidence — confidence score (0.0 to 1.0) // data.hand_detected — boolean, true if hand visible // data.landmarks — object with hand/pose landmark coordinates // data.probabilities — array of probabilities per class });

🌐 REST API Endpoints

GET /api/health Check server status

Response

{ "status": "healthy" }
GET /api/gestures Get supported gestures

Response

{ "gestures": ["thanks", "i love you"], "count": 2 }
POST /api/translate Translate gesture text to another language

Request Body

{ "text": "thanks", "target_language": "es" }

Parameters

ParameterTypeDescription
textstringThe gesture text to translate
target_languagestringISO language code: en, es, fr, ar, hi, ja

Response

{ "translated": "Gracias" }
POST /api/contribute Submit crowdsourced keypoint data

Request Body

{ "gesture": "thanks", "confidence": 0.999, "timestamp": "2026-05-10T12:00:00.000Z" }

Response

{ "status": "saved", "total": 42 }
GET /api/contributions/stats Get contribution counts per gesture

Response

{ "thanks": 25, "i love you": 17 }
GET /api/contributions/recent Get last 50 contributions

Response

[ { "gesture": "thanks", "confidence": 0.9994, "timestamp": "2026-05-10T12:00:00.000Z" } ]
GET /api/contributions/export Export all contributions as CSV

Response

Content-Type: text/csv Content-Disposition: attachment; filename=contributions.csv timestamp,gesture,confidence 2026-05-10T12:00:00,thanks,0.9994 2026-05-10T12:01:00,i love you,0.9998
GET /api/analytics Get session and environmental analytics

Response

{ "total_sessions": 42, "total_gestures": 187, "gesture_counts": { "thanks": 95, "i love you": 92 }, "hourly_detections": { "9": 12, "10": 25, "14": 30 }, "avg_confidence": 99.7, "most_popular_gesture": "thanks", "daily_sessions": { "2026-05-10": 5 } }
